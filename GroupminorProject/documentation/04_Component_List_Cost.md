## Component Specifications

### ESP32 Development Board
- **Model**: ESP32-WROOM-32
- **Features**: WiFi 802.11 b/g/n, Bluetooth 4.2, 32-bit dual-core processor
- **GPIO Pins**: 34 digital pins, 12 analog pins
- **Power**: 3.3V, 5V tolerant I/O

### Flow Sensor (YF-S201)
- **Flow Rate Range**: 1-30 L/min
- **Accuracy**: ±3%
- **Output**: Pulse signal (450 pulses/L)
- **Operating Voltage**: 5V DC
- **Max Pressure**: 1.75 MPa

### Vibration Sensor (MPU6050)
- **Accelerometer Range**: ±2g, ±4g, ±8g, ±16g
- **Gyroscope Range**: ±250°/s to ±2000°/s
- **Interface**: I2C
- **Operating Voltage**: 3.3V or 5V

### Current Sensor (ACS712)
- **Current Range**: ±30A
- **Sensitivity**: 66mV/A
- **Output**: Analog voltage (0-5V)
- **Accuracy**: ±1.5%

### Temperature Sensor (LM35)
- **Temperature Range**: -55°C to +150°C
- **Accuracy**: ±0.5°C
- **Output**: 10mV/°C
- **Linear Output**

### pH Sensor Module
- **pH Range**: 0-14
- **Accuracy**: ±0.1 pH
- **Response Time**: < 1 minute
- **Operating Temperature**: 0-60°C

### Water Level Sensor
- **Type**: Ultrasonic HC-SR04 or Resistive
- **Range**: 2-400 cm (ultrasonic) or 0-5 cm (resistive)
- **Accuracy**: ±3mm (ultrasonic)

## Cost Optimization Tips

1. Buy components in bulk if building multiple prototypes
2. Use local suppliers for basic components (wires, resistors)
3. Consider used/second-hand components for testing
4. Group orders to save on shipping
5. Look for student discounts on educational platforms
